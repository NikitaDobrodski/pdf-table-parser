# PDF Table Parser (pdf-table-parser)
Сервис на **FastAPI**, который извлекает таблицы из PDF и возвращает результат в **JSON / CSV / XLSX**. <!-- описание репозитория -->


---

## Структура проекта


pdf-table-parser/
├─ backend/
│  ├─ app/
│  │  ├─ __init__.py
│  │  ├─ main.py        # FastAPI API
│  │  ├─ parser.py      # извлечение таблиц из PDF
│  │  └─ exporters.py   # экспорт XLSX (openpyxl)
│  ├─ requirements.txt
│  └─ Dockerfile
└─ docker-compose.yml

1. Клонирование репозитория 

```git clone https://github.com/OkulusDev/Oxygen.git```

2. Переход в директорию Oxygen

```cd Oxygen```

3. Создание виртуального окружения

```python3 -m venv venv```

4. Активация виртуального окружения

```source venv/bin/activate```

5. Установка зависимостей

```pip3 install -r requirements.txt```

6. Запуск скрипта для демонстрации возможностей Oxygen

```python3 oxygen.py --help```
